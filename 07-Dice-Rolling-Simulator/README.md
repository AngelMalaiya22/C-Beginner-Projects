# Dice Rolling Simulator

## Overview

A simple C program that simulates rolling a six-sided dice. Each roll generates a random number between 1 and 6, and the user can continue rolling as many times as they like.

## Features

* Simulates a six-sided dice
* Random number generation
* Supports multiple dice rolls
* Interactive menu
* Beginner-friendly implementation

## Technologies Used

* C

## How to Run

1. Compile the program using a C compiler.
2. Run the executable.
3. View the dice roll result.
4. Choose whether to roll again.
