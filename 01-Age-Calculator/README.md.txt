# Age Calculator

## Overview

A simple C program that calculates a person's age based on their date of birth and the current date entered by the user. The program accurately determines the age in years.

## Features

* Calculates age in years
* Accepts date of birth and current date as input
* Accurate age calculation
* Beginner-friendly implementation

## Technologies Used

* C

## How to Run

1. Compile the program using a C compiler.
2. Run the executable.
3. Enter your date of birth and today's date in **DD MM YYYY** format.
4. View your calculated age.
