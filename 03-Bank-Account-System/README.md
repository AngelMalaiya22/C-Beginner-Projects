# Bank Account System

## Overview

A simple C program that demonstrates a basic bank account management system using structures. Users can check their account balance, deposit money, withdraw money, and manage their account through a menu-driven interface.

## Features

* Create a bank account
* Check account balance
* Deposit money
* Withdraw money
* Uses C structures
* Menu-driven interface

## Technologies Used

* C

## How to Run

1. Compile the program using a C compiler.
2. Run the executable.
3. Enter the account holder's name.
4. Choose an option from the menu to manage the account.
