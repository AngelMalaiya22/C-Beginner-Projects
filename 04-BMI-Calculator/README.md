# BMI Calculator

## Overview

A simple C program that calculates the Body Mass Index (BMI) using the user's weight and height. The program also categorizes the BMI based on standard health ranges.

## Features

* Calculate Body Mass Index (BMI)
* Classify BMI into health categories
* Displays BMI up to two decimal places
* Beginner-friendly implementation

## Technologies Used

* C

## How to Run

1. Compile the program using a C compiler.
2. Run the executable.
3. Enter your weight in kilograms and height in meters.
4. View your BMI and health category.
