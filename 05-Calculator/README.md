# Calculator

## Overview

A simple calculator program written in C that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Handles division by zero
* Menu-driven interface

## Technologies Used

* C

## How to Run

1. Compile the program using a C compiler.
2. Run the executable.
3. Select an operation and enter two numbers.
4. View the calculated result.
