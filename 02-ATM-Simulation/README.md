# ATM Simulation

## Overview

A simple C program that simulates basic ATM operations. Users can check their account balance, deposit money, withdraw money, and exit the application through a menu-driven interface.

## Features

* Check account balance
* Deposit money
* Withdraw money
* Prevents overdrawing the account
* Menu-driven interface

## Technologies Used

* C

## How to Run

1. Compile the program using a C compiler.
2. Run the executable.
3. Select an option from the ATM menu.
4. Perform banking operations until you choose to exit.
