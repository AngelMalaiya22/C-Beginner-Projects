# Coin Toss Simulator

## Overview

A simple C program that simulates tossing a coin. The program randomly displays either **Heads** or **Tails**, allowing users to perform multiple coin tosses.

## Features

* Randomly generates Heads or Tails
* Supports multiple coin tosses
* Interactive and beginner-friendly
* Uses random number generation

## Technologies Used

* C

## How to Run

1. Compile the program using a C compiler.
2. Run the executable.
3. View the coin toss result.
4. Choose whether to toss the coin again.
